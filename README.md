# Data-youth-campus-hackathon

[데이터 청년 캠퍼스](https://dataonair.or.kr/bigjob/) 프로그램에서 진행한 해커톤
> 2023.06.26 ~ 2023.08.31 기간동안 교육 및 프로젝트 진행
> 활동기간 중 수원대 데청캠 인원 대상으로 해커톤 진행  
--------------------------------------------------------------------------------------------------------------------------------------------
## 데이터 설명
데이콘에서 제공한 csv데이터 활용  
> 아파트 지역, 지번, 아파트 이름, 공간, 거래년월, 거래일자, 실거래 가격, 층수, 건설완료년도등 포함
